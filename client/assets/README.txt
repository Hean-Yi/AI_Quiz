Please place your App Icon and Splash Screen images here:

1. **Icon**: 
   - Filename: `icon.png` (or `icon.jpg`)
   - Recommended Size: 1024 x 1024 pixels
   - This will be used for the App Icon on the home screen.

2. **Splash Screen** (Optional):
   - Filename: `splash.png` (or `splash.jpg`)
   - Recommended Size: 2732 x 2732 pixels
   - This will be shown while the app is loading.

After placing the files, run the build script to generate the APK.
